# Checklist CEFR A0 -> A1 (observable)

Marcar fecha cuando se logra de forma consistente (no solo una vez).

## A0.0 -> A0+

- [ ] Puede dar nombre, país y ciudad en 3 frases simples.
- [ ] Puede deletrear datos básicos (nombre/apellido).
- [ ] Puede preguntar y decir la hora de forma básica.
- [ ] Puede pedir algo en tienda con frase completa.
- [ ] Puede decir 5 acciones de rutina diaria en presente.

## A0+ -> A1.1 temprano

- [ ] Puede manejar presentación de 1 minuto sin cambiar al español.
- [ ] Puede hacer preguntas con `wo`, `wann`, `wie`, `warum` en contexto.
- [ ] Puede describir vivienda/barrio con 6-8 frases.
- [ ] Puede resolver una compra simple (cantidades, precio, preferencia).
- [ ] Puede pedir/confirmar una cita (fecha/hora/motivo).
- [ ] Puede describir síntomas básicos y entender instrucciones simples.
- [ ] Puede explicar rutina laboral básica (horario, tareas simples).
- [ ] Puede completar oralmente datos típicos de un formulario.

## A1.1 consolidación funcional

- [ ] Mantiene diálogo de 3-5 min sobre vida diaria con pausas razonables.
- [ ] Reacciona con frases funcionales de reparación: `Wie bitte?`, `Langsam, bitte.`, `Ich habe nicht verstanden.`
- [ ] Usa pasado muy básico en expresiones memorizadas (`war`, `hatte`) cuando aplica.
- [ ] Comete errores, pero sigue siendo comprensible para interlocutor paciente.
