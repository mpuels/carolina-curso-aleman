# Diagnóstico inicial

Este bloque permite estimar el nivel real de inicio y decidir si arrancar exactamente en `A0.0`, `A0+` o `A1.1-ready`.

## Pasos

1. Aplicar `placement-test.md` en una sola sesión (45-60 min).
2. Corregir con `placement-answer-key.md`.
3. Clasificar con `level-decision-rubric.md`.
4. Ajustar intensidad semanal:
   - `A0.0`: seguir el plan tal cual.
   - `A0+`: mantener plan, aumentar producción oral.
   - `A1.1-ready`: acelerar en semanas 3-6.

## Condiciones recomendadas

- Realizar la prueba sin diccionario ni traductor.
- Tomar notas de pronunciación y bloqueos.
- Grabar (audio) la parte oral para comparar progreso en semana 3 y 6.
